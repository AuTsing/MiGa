# MiGa 米咖

MiGa (米咖) 是基于米家 API 开发的第三方应用，为手表开发，旨在手表实现米家 APP 的功能。

## 功能

-   ✅ 浏览设备列表
-   ✅ 浏览智能场景列表
-   ✅ 手动触发智能场景
-   ✅ 收藏智能场景
-   ✅ 通过功能块(卡片)触发智能场景
-   ✅ 通过复杂功能(表盘)触发智能场景
-   [ ] 设备控制
